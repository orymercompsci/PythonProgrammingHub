6. Car Journey

input() float() print()

Write a program to calculate the cost of a car journey.

  • Prompt for miles to be driven.
  • Prompt for the car's miles per gallon (MPG).
  • Prompt for the current price of 1 gallon of fuel.
  • Calculate gallons needed: distance / mpg.
  • Calculate total cost: gallons_needed * fuel_price.
  • Output the gallons needed and the total cost with suitable messages.

Example Output:

Miles to drive: 200
Car MPG: 40
Price per gallon: 6.50
You will need 5.0 gallons of fuel.
The total cost of the trip will be £ 32.5.

Incorrect password.

Solution 
distance = float(input("Miles to drive: "))
mpg = float(input("Car MPG: "))
fuel_price = float(input("Price per gallon: "))
gallons_needed = distance / mpg
total_cost = gallons_needed * fuel_price
print("You will need", gallons_needed, "gallons of fuel.")
print("The total cost of the trip will be £", total_cost)

8. Piggy Bank

input() int() print()

Calculate the total value of coins in a piggy bank.

  • Prompt the user for the number of each coin type (1p, 2p, 5p, 10p, 20p, 50p, £1, £2).
  • Calculate the total value in pounds.
  • Display the total value to the user.

Example Output:

How many 1p coins?: 100
How many 2p coins?: 50
...
Total value in your piggy bank: £ 25.75

Incorrect password.

Solution

total = pennies + two_pences + five_pences + ten_pences + twenty_pences + fifty_pences + one_pounds + two_pounds
print("Total value in your piggy bank: £", total)

9. Give me the Discount

input() float() print()

Simulate applying a discount to an item's price.

  • User enters the full price of an item.
  • User enters the discount percentage.

  • Apply the discount to get the final price.
  • Output the discounted price.

Enter full price: 75

The discounted price is £ 60.0

Incorrect password.

Solution

13. Secret Agent ID Generator

input() int() len() print()

Create a three-part Agent ID from a secret number.

  • Ask user for their secret number.
  • Part 1: Secret number integer divided (//) by its number of digits.
  • Part 2: Remainder of the secret number divided (%) by its number of digits.
  • Part 3: The number of digits in the secret number.
  • Print the ID in the format: PART1-PART2-PART3.

Example Output:

Enter your secret number: 98172
Your Agent ID is: 19634 - 2 - 5

Incorrect password.

Solution 
secret_number_str = input("Enter your secret number: ")
secret_number_int = int(secret_number_str)
num_digits = len(secret_number_str)

part1 = secret_number_int // num_digits
part2 = secret_number_int % num_digits
part3 = num_digits

print("Your Agent ID is:", part1, "-", part2, "-", part3)

15. Pig Latin Translator

input() print() Slicing [:]

Translate a word into Pig Latin using a specific method.

  1. Input a word from the user.
  2. Remove the first letter (e.g., Banana -> anana).
  3. Add a hyphen (anana -> anana-).
  4. Put the first letter after the hyphen (anana- -> anana-B).
  5. Add "ay" at the end (anana-B -> anana-Bay).
  6. Output the translated word.

Example Output:

Enter a word: Python
ython-Pay

Incorrect password.

Solution 
word = input("Enter a word: ")
first_letter = word[0]
rest_of_word = word[1:]
pig_latin_word = rest_of_word + "-" + first_letter + "ay"
print(pig_latin_word)
