Sequence

In programming, sequence refers to the execution of code statements one after another, in the order they are written. Python reads and runs your code from top to bottom, like reading a book. Each line is an instruction that is followed before moving to the next.

Challenge 12: Secret Agent ID Number Generator

  • Calculate a three-part Agent ID using the formula provided.

The ID Formula:
Part 1: The secret number integer divided (//) by its number of digits.
Part 2: The remainder of the secret number divided (%) by its number of digits.
Part 3: The number of digits in the secret number.

Hover over a command to see its description.

str() Converts a value into a string.
// (Floor Division) Divides and returns the whole number result, discarding the remainder.
% (Modulus) Divides and returns only the remainder.
print() Used to output text and variables to the screen.

Example Screen Output:

Enter your secret number: 98172
Your Secret Agent ID is: 19634-2-5

Testing Table

Test Inputs Expected Outputs
Test 1 (Valid) Secret Number: 123 Your Secret Agent ID is: 41-0-3
Test 2 (Valid) Secret Number: 8000 Your Secret Agent ID is: 2000-0-4
Test 3 (Boundary) Secret Number: 9 Your Secret Agent ID is: 9-0-1

View Solution

Enter the password to see the solution. Hint: the initials of the challenge title.

Incorrect password. Please try again.

Example Solution:

# Get the secret number as a string to easily find its length

# Convert the string to an integer for calculations

# Get the number of digits using len() on the string version

part2 = secret_number_int % num_digits
