Sequence

In programming, sequence refers to the execution of code statements one after another, in the order they are written. Python reads and runs your code from top to bottom, like reading a book. Each line is an instruction that is followed before moving to the next.

Challenge 1: Total Money

Write a program that will:

  • Assign the name "Fred" (a string) to a variable called name.
  • Assign the number 15 (an integer) to a variable called pounds.
  • Assign the number 10 (an integer) to a variable called pocket_money.
  • Add pocket_money to pounds and store the result.
  • Output a message to the screen showing the total amount of money Fred has.

Expected Screen Output:

Fred has 25 pounds

Example Solution:

# Assign the string "Fred" to the variable 'name'
name = "Fred"

# Assign the integer 15 to the variable 'pounds'
pounds = 15

# Assign the integer 10 to the variable 'pocket_money'
pocket_money = 10

# Add the value of 'pocket_money' to 'pounds' and update 'pounds' with the new value (15 + 10 = 25)
pounds = pounds + pocket_money

# Print the final sentence, combining the variables and strings.
# The print function automatically adds spaces between items.
print(name, "has", pounds, "pounds")

Alternate Solution:

This version is often preferred because it uses a new variable for the total, which can make the code easier to read and understand.

# Assign the string "Fred" to the variable 'name'
name = "Fred"

# Assign the integer 15 to the variable 'pounds'
pounds = 15

# Assign the integer 10 to the variable 'pocket_money'
pocket_money = 10

# Create a new variable 'total' to store the sum of 'pounds' and 'pocket_money'
total = pounds + pocket_money

# Print the final sentence using the new 'total' variable
print(name, "has", total, "pounds")

Write a program that will create a randomised addition sum and print it to the screen.

  • Generate two random numbers between 1 and 100.

# Import the 'random' module, which contains functions for generating random numbers.
import random

# Generate a random integer between 1 and 100 (inclusive) and store it in num1.
num1 = random.randint(1, 100)

# Generate a second random integer and store it in num2.
num2 = random.randint(1, 100)

# Print the two numbers in the desired format.
print(num1, "+", num2, "= ?")

Challenge 3: Car Journey

Write a program that calculates the cost of a car journey.

  • Prompt the user to enter the distance of the trip (in miles).
  • Prompt the user for their car's miles per gallon (MPG).
  • Prompt the user for the current price of one gallon of fuel.
  • Calculate the number of gallons needed and the total cost of the trip.
  • Display the results to the user in a clear, formatted message.

Example Screen Output:

Enter the distance to be driven (miles): 100
Enter the car's MPG: 50
Enter the price per gallon (£): 1.50

Gallons needed: 2.0
Total cost of the trip: £3.00

Example Solution:

# The input() function gets data from the user, but it always returns it as a string.
# We use float() to convert the string input into a floating-point number (a number with a decimal place).

# Get the distance from the user and convert it to a float.
distance = float(input("Enter the distance to be driven (miles): "))

# Get the car's MPG from the user and convert it to a float.
mpg = float(input("Enter the car's MPG: "))

# Get the fuel price from the user and convert it to a float.
fuel_price = float(input("Enter the price per gallon (£): "))

# Calculate the number of gallons needed for the trip.
gallons_needed = distance / mpg

# Calculate the total cost of the fuel for the trip.
total_cost = gallons_needed * fuel_price

# Print the results. An f-string is used here to format the output.
# {gallons_needed} is a placeholder that gets replaced by the value of the variable.
# :.1f formats the gallons to 1 decimal place.
# :.2f formats the cost to 2 decimal places, which is standard for currency.
print(f"\nGallons needed: {gallons_needed:.1f}")
print(f"Total cost of the trip: £{total_cost:.2f}")

Iteration means repeating a block of code multiple times. This is often called "looping". It's useful for when you want to perform the same action on a collection of items or repeat an action a specific number of times. Python uses for and while loops for iteration.
