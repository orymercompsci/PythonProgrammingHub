Sequence

In programming, sequence refers to the execution of code statements one after another, in the order they are written. Python reads and runs your code from top to bottom, like reading a book. Each line is an instruction that is followed before moving to the next.

Programming Challenge: Total Money

Write a program that will:

  • Assign the name "Fred" (a string) to a variable called name.
  • Assign the number 15 (an integer) to a variable called pounds.
  • Assign the number 10 (an integer) to a variable called pocket_money.
  • Add pocket_money to pounds and store the result.
  • Output a message to the screen showing the total amount of money Fred has.

Expected Output:

Fred has 25 pounds

Incorrect password. Please try again.

Example Solution:

# Assign values to variables
name = "Fred"
pounds = 15
pocket_money = 10

# Calculate the total amount
total_pounds = pounds + pocket_money

# Print the final message
# We use an f-string to easily combine text and variables
print(f"{name} has {total_pounds} pounds")

Iteration

Iteration means repeating a block of code multiple times. This is often called "looping". It's useful for when you want to perform the same action on a collection of items or repeat an action a specific number of times. Python uses for and while loops for iteration.
